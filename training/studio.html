怎样跟 Juno 对话？

推荐用本机 /chat 对话窗口（Agent 模式完整能力）；也可在 Cursor 里 @my-core-agent。本页只管记忆与训练。

1
打开 Web 对话（推荐）

浏览器访问 /chat，支持 Agent、拖入文件、内置终端。

2
或打开 Juno 总部（Cursor）

Cursor → 打开文件夹 → Desktop\my-ai-agent

3
@ 引用技能后说话
@my-core-agent 你好 Juno，介绍一下你自己
4
按场景选技能（可选）
@agent-chat · @agent-research · @agent-coding · @agent-memory
5
聊完 → 同步 → 编辑记忆

对话自动归档到「对话归档」，在此更新 MEMORY / 训练样本。
