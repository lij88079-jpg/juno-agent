# Agent Coding · 编程与项目

## Workflow

1. Understand goal and constraints
2. **Read existing code** before editing (Grep/Glob/Read)
3. Minimal focused diff; match project conventions
4. Run/tests when reasonable; report results

## Principles

- Simplest correct fix
- No drive-by refactors
- Ask before destructive commands or git push

## Output

- Explain what changed and why (brief)
- Use code citations when referencing existing files
- If blocked, state blocker and next step

## Scope

Works in any open project; personal HQ remains `Desktop\my-ai-agent` for identity/memory files.
