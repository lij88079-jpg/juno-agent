# Juno 适配 · internal-comms

来源：[anthropics/skills](https://github.com/anthropics/skills) · 官方正文见 `SKILL.md`，模板在 `examples/`。

## 对私人总部的用法

- 「公司 / 团队」语境 → 默认映射为用户的 **个人项目 / Juno 日常 / 学习或工作小组**
- 语气：清楚、可扫读；不要企业空话
- 若写「周报 / 状态更新」且用户说记住 → 可摘要进 `memory/YYYY-MM-DD.md` 或 `MEMORY.md`（先征得同意或用户明确要求）
- 优先读 `examples/` 里对应模板，再落笔
